Project Title: Emergency Contact App


About the Project
The Emergency Contact App is an Android application developed using Android Studio and Java. This app is designed to provide users with a quick and reliable way to contact help during emergency situations.
In critical moments, manually searching for contacts can be time-consuming. This app simplifies the process by allowing users to store important emergency contacts and access them instantly through a user-friendly interface. With features like one-tap calling and an SOS button, users can quickly reach out for help when needed.

Features
Save Emergency Contacts
Add, edit, and delete important contacts.

One-Tap Calling
Instantly call saved contacts with a single tap.

SOS Button
Quickly call a primary emergency contact or send alerts.

Location Sharing (Optional)
Share real-time location via SMS with selected contacts.

Simple User Interface
Clean and easy-to-use design for quick access.

Tech Stack
Programming Language: Java
IDE: Android Studio
Platform: Android
APIs: Android SDK, Phone Call & SMS APIs, GPS/Location Services
